## 📤 Contribution Guidelines

欢迎所有同学贡献他们的笔记和资源！请将你的笔记按以下格式提交：
1. 课程名称（包含授课教师、选课时间）
2. 资源类型（电子版课程笔记、答疑视频、帖子链接等）
3. 你的姓名（将会被在首页致谢）
#### Attention：单份笔记不超过25M，方便上传；若笔记过大请拆分处理
